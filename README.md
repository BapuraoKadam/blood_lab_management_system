# 🧪 Online Blood Test Lab Management System

## 📌 Project Description
This project is a web-based application developed using Spring Boot that allows patients to book lab tests online, view reports, and manage their bookings.

## 🚀 Features
- User Registration & Login
- Book Blood Tests
- View Booking History
- Admin Panel to Manage Tests
- Report Generation (PDF)
- Search & Filter Tests

## 🛠️ Technologies Used
- Java (Spring Boot)
- Hibernate / JPA
- MySQL Database
- Thymeleaf
- HTML, CSS, Bootstrap

## 📂 Project Structure
- Controller Layer
- Service Layer
- Repository Layer
- Entity Classes
- Templates (HTML)
