# 🧪 Online Blood Test Lab Management System

## 📌 Project Description
This is a web-based Blood Lab Management System built using Spring Boot.  
It allows patients to book blood tests online, view reports, and manage their bookings efficiently.  
Admins can manage tests, bookings, and generate reports.

---

## 🚀 Features

### 👤 User Module
- User Registration & Login
- Book Blood Tests
- View Booking History
- Download Reports (PDF)

### 🔐 Admin Module
- Admin Dashboard
- Add / Edit / Delete Tests
- Manage Bookings
- Upload & Generate Reports

## ⚙️ Installation & Setup

1. Clone the repository  
   git clone https://github.com/BapuraoKadam/blood_lab_management_system.git  

2. Open project in IntelliJ / Eclipse  

3. Configure MySQL in application.properties  
   spring.datasource.url=jdbc:mysql://localhost:3306/blood_lab  
   spring.datasource.username=root  
   spring.datasource.password=your_password  

4. Run the project  
   mvn spring-boot:run  

5. Open in browser  
   http://localhost:8080
